# mycraps
Minecraft clone. Uses lpaTPGEngine.

## Compiling
compile with this command (in mycraps/examplemc)
```bash
g++ examplemc.cpp -std=c++26 -mcmodel=large -O3 -lSDL2 -lSDL2_image -lSDL2_ttf -o testmc
```

The binary name can be anything
